# Last.fm Music Widget (LotusLabs Adaptation)

This is a LotusLabs adaptation of Nutty's widget, updated to use Last.fm integration instead of Spotify to free up your one Spotify developer application.

Thanks Nutty for the original widget.

## Original Widget

Link: `https://github.com/nuttylmao/spotify-widget`

## What Changed

- Replaced Spotify auth/API flow with Last.fm API support
- Uses URL params: `apikey` and `username`
- Added a `configure/` page to generate/launch the widget URL
- Added optional settings for `duration` and `hideAlbumArt`
